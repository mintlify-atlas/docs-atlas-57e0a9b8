## Installation

```bash
npx @cloudflare/kumo add meter
```

## Import

```tsx
import { Meter } from "@cloudflare/kumo";
```

## Usage

```tsx
<Meter label="Storage used" value={65} />
```

## Props

<ParamField path="label" type="string" required>
  Label text displayed above the meter track.
</ParamField>

<ParamField path="value" type="number" required>
  Current value of the meter.
</ParamField>

<ParamField path="max" type="number" default="100">
  Maximum value of the meter.
</ParamField>

<ParamField path="min" type="number" default="0">
  Minimum value of the meter.
</ParamField>

<ParamField path="customValue" type="string">
  Custom formatted value text (e.g. "750 / 1,000") displayed instead of percentage.
</ParamField>

<ParamField path="showValue" type="boolean" default="true">
  Whether to display the percentage value next to the label.
</ParamField>

<ParamField path="trackClassName" type="string">
  Additional CSS classes for the track (background bar).
</ParamField>

<ParamField path="indicatorClassName" type="string">
  Additional CSS classes for the indicator (filled bar).
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for the root container.
</ParamField>

## Built on Base UI

Meter is built on [`@base-ui/react/meter`](https://base-ui.com/components/meter) and uses the following sub-components internally:

- `Meter.Root` — Container with value/min/max
- `Meter.Label` — Accessible label
- `Meter.Value` — Formatted percentage display
- `Meter.Track` — Background bar
- `Meter.Indicator` — Filled progress bar

## Accessibility

- Uses semantic `<meter>` element via Base UI
- `aria-label` provided by `Meter.Label`
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` automatically set
- Percentage displayed visually and announced to screen readers

## Styling

Meter uses semantic tokens:

- **Label**: `text-xs text-kumo-strong`
- **Value**: `text-sm font-medium text-kumo-default tabular-nums`
- **Track**: `h-2 w-full rounded-full bg-kumo-fill`
- **Indicator**: `rounded-full bg-linear-to-r from-kumo-brand via-kumo-brand to-kumo-brand`
- **Transition**: `transition-[width] duration-300 ease-out`
