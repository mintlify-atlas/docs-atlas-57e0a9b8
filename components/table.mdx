title: Table

## Import

```tsx
import { Table } from "@cloudflare/kumo";
```

## Usage

```tsx
<Table>
  <Table.Header>
    <Table.Row>
      <Table.Head>Name</Table.Head>
      <Table.Head>Status</Table.Head>
    </Table.Row>
  </Table.Header>
  <Table.Body>
    <Table.Row>
      <Table.Cell>Worker A</Table.Cell>
      <Table.Cell>Active</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table>
```

### Table with Checkboxes

```tsx
const [selected, setSelected] = useState(new Set());
const allSelected = selected.size === rows.length;
const indeterminate = selected.size > 0 && !allSelected;

const toggleAll = () => {
  if (allSelected) {
    setSelected(new Set());
  } else {
    setSelected(new Set(rows.map(r => r.id)));
  }
};

const toggle = (id) => {
  const newSelected = new Set(selected);
  if (newSelected.has(id)) {
    newSelected.delete(id);
  } else {
    newSelected.add(id);
  }
  setSelected(newSelected);
};

<Table>
  <Table.Header>
    <Table.Row>
      <Table.CheckHead 
        checked={allSelected} 
        indeterminate={indeterminate}
        onValueChange={toggleAll} 
      />
      <Table.Head>Name</Table.Head>
      <Table.Head>Status</Table.Head>
    </Table.Row>
  </Table.Header>
  <Table.Body>
    {rows.map((row) => (
      <Table.Row 
        key={row.id} 
        variant={selected.has(row.id) ? "selected" : "default"}
      >
        <Table.CheckCell 
          checked={selected.has(row.id)} 
          onValueChange={() => toggle(row.id)} 
        />
        <Table.Cell>{row.name}</Table.Cell>
        <Table.Cell>{row.status}</Table.Cell>
      </Table.Row>
    ))}
  </Table.Body>
</Table>
```

## Components

### Table (Root)

```tsx
<Table layout="auto">
```

#### Props

<ParamField path="layout" type="'auto' | 'fixed'" default="auto">
Table layout algorithm:
- `auto` — Columns resize based on content
- `fixed` — Equal-width columns, controlled via `<colgroup>`
</ParamField>

<ParamField path="className" type="string">
Additional CSS classes.
</ParamField>

### Table.Header

Wrapper for table header rows:

```tsx
<Table.Header>
  <Table.Row>
    <Table.Head>Name</Table.Head>
  </Table.Row>
</Table.Header>
```

Renders a `<thead>` element.

### Table.Head

Header cell:

```tsx
<Table.Head>Column Name</Table.Head>
```

Renders a `<th>` element with:
- Font: `font-semibold`
- Background: `bg-kumo-base`
- Border: `border-b border-kumo-fill`
- Padding: `p-3`
- Group: `group` class for resize handle visibility

### Table.Body

Wrapper for table body rows:

```tsx
<Table.Body>
  <Table.Row>
    <Table.Cell>Data</Table.Cell>
  </Table.Row>
</Table.Body>
```

Renders a `<tbody>` element.

### Table.Row

Table row with optional variant:

```tsx
<Table.Row variant="default">
```

#### Props

<ParamField path="variant" type="'default' | 'selected'" default="default">
Row visual style:
- `default` — Normal row appearance
- `selected` — Highlighted row with `bg-kumo-tint`
</ParamField>

### Table.Cell

Standard table cell:

```tsx
<Table.Cell>Content</Table.Cell>
```

Renders a `<td>` element with:
- Border: `border-b border-kumo-fill` (except last row)
- Padding: `p-3`
- Color: `text-kumo-default`

### Table.CheckCell

Cell with checkbox for row selection:

```tsx
<Table.CheckCell 
  checked={isSelected} 
  onValueChange={toggleSelection}
  label="Select row"
/>
```

#### Props

<ParamField path="checked" type="boolean">
Whether the checkbox is checked.
</ParamField>

<ParamField path="indeterminate" type="boolean">
Whether the checkbox is in indeterminate state.
</ParamField>

<ParamField path="onValueChange" type="(checked: boolean) => void">
Callback when checkbox value changes.
</ParamField>

<ParamField path="label" type="string" default="Select row">
Accessible label for the checkbox.
</ParamField>

<ParamField path="disabled" type="boolean">
Whether the checkbox is disabled.
</ParamField>

### Table.CheckHead

Header cell with checkbox for "select all":

```tsx
<Table.CheckHead 
  checked={allSelected}
  indeterminate={someSelected}
  onValueChange={toggleAll}
  label="Select all rows"
/>
```

Same props as `Table.CheckCell`, but default label is "Select all rows".

### Table.Footer

Wrapper for table footer rows:

```tsx
<Table.Footer>
  <Table.Row>
    <Table.Cell>Total</Table.Cell>
  </Table.Row>
</Table.Footer>
```

Renders a `<tfoot>` element.

### Table.ResizeHandle

Draggable handle for column resizing:

```tsx
<Table.Head>
  Column Name
  <Table.ResizeHandle />
</Table.Head>
```

Renders a `<button>` with:
- Position: `absolute top-0 right-0`
- Visibility: `invisible group-hover:visible`
- Cursor: `cursor-col-resize`
- Width: `w-[10px]` hit area
- Visual: `h-5 w-[2px] rounded bg-kumo-ring`

## Layout Variants

### Auto Layout (default)

```tsx
<Table layout="auto">
```

Columns automatically size based on content. Cells can wrap.

### Fixed Layout

```tsx
<Table layout="fixed">
  <colgroup>
    <col style={{ width: '40%' }} />
    <col style={{ width: '30%' }} />
    <col style={{ width: '30%' }} />
  </colgroup>
  {/* ... */}
</Table>
```

Columns have equal width unless specified via `<colgroup>`. Content truncates if too wide.

## Row Variants

### Default

```tsx
<Table.Row variant="default">
```

Standard row appearance.

### Selected

```tsx
<Table.Row variant="selected">
```

Highlighted row with `bg-kumo-tint` background.

## Styling

### Base Table Styles

```tsx
className="w-full text-left text-kumo-default"
```

### Cell Borders

Applied via CSS selectors:

```tsx
[&_tr_td]:border-b [&_tr_td]:border-kumo-fill [&_tr:last-child_td]:border-b-0
```

- All cells: Bottom border
- Last row: No bottom border

### Cell Padding

```tsx
[&_tr_td]:p-3
```

All cells have `p-3` padding.

### Header Styles

```tsx
[&_tr_th]:border-b [&_tr_th]:border-kumo-fill [&_tr_th]:p-3 [&_tr_th]:font-semibold [&_tr_th]:bg-kumo-base
```

- Border: Bottom border
- Padding: `p-3`
- Font: `font-semibold`
- Background: `bg-kumo-base`

## Checkbox Integration

CheckCell and CheckHead use the Checkbox component internally:

- Checkbox is positioned with `relative` and `before:absolute before:-inset-3` to expand hit area
- Width: `w-10` column width
- Leading: `leading-none` for vertical centering

## Accessibility

- Uses semantic HTML table elements (`<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<tr>`, `<th>`, `<td>`)
- Header cells use `<th>` for screen reader navigation
- Checkboxes include `aria-label` (customizable)
- ResizeHandle includes `aria-label="Resize column"`
- Left-aligned text for readability (`text-left`)

## Notes

- Table uses CSS selectors for styling cells to avoid prop drilling
- Checkbox hit area extends beyond visible checkbox via `before` pseudo-element
- ResizeHandle requires manual drag implementation (component provides UI only)
- For sortable headers, add onClick handlers to Table.Head and include sort icons
- For sticky headers, add `sticky top-0 z-10` to Table.Header className
